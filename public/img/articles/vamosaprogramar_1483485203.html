Introducing Pts Fashion Store Prestashop theme

Pts Fashion Store is a responsive Prestashop theme for fashion. The Prestashop theme is developed with SASS - Syntactically Awesome Style Sheets, it integrates Bootstrap 3, Font Awesome, supports HTML5 and CSS3.

Pts Fashion Store supports Prestashop modules such as:

  • Pts Display Categories Information
  • Pts Products Carousel Block
  • Pts Static Content Manager
  • Pts Top Rating products
  • Pts Manufacturers Block
  • Pts Twitter Modules
  • Pts Related Products Block
  • Pts Map Locator

You can replicate our Demo site and build up your own E-commerce website in some simple steps with our Quickstart package.

System Requirement

Prestashop requires certain technical requirements to be met for the store to operate properly. First, a web server must be created to make the Prestashop store publicly available on the web. Domain names and hosting services can easily be purchased for an affordable price.

When selecting a hosting service, you should check to see that these server requirements are provided and installed on their web servers:

  • Supported operating system: Windows, Mac and Linux
  • Wamp (for Windows) or Xampp (for Windows and Mac)
  • PHP 5.1+ installed and enabled
  • MySQL 5.0+ installed with a database created

PHP configuration:

  • "memory_limit" should be at least "64M"
  • "file_max_upload_size" to "16M"
  • “allow_url_fopen” should be set to “on”
  • "register global" should be set to "off"
  • "safe_mode" should be set to "off"

Optional PHP extensions:

  • GD
  • CURL
  • SimpleXML
  • SOAP

Permissions:

  • Permissions for all files should be set to "664"
  • Permissions for all folders should be set to "755"

If you don't know how to check these requirements, ask your hosting provider.

Theme Installation

Installing a PrestaBrain themes is super easy. Before you install this theme:

  1. Read our complete instruction to install theme
  2. Make sure you have installed and running the correct version of Prestashop: Prestashop 1.6.x
  3. Take a FULL BACKUP for your site

We recommend to use this theme on a new, clean installation or existing shops without custom core modifications.

Steps to install Fashion Store Theme manually:

  1. Navigate to PREFERENCES → Themes in your Prestashop admin dashboard.
  2. Click the Add New Theme button at the top of the page and go for the Import option. ( you can import from your computer, from web or from FTP)
  3. Click Add file from "IMPORT FROM YOUR COMPUTER" option. For the file upload, pick Theme package/ name_theme.zip in the theme package and click Save.

Please watch our video guide to know more.

Theme Configuration

Supported themes/layouts

Pts Fashion Store has only main theme/layout.

Activate Theme

Once the theme is uploaded, in your Prestashop admin dashboard, navigate to: PREFERENCES > Themes.  Go to Select a theme for the "Your Store Name" shop, click Use this theme as you want.

Check module before installing.

Theme has been successfully actived. You may have to regenerate images to fit with this new theme.

Setup Logo

To configure your logo, navigate to PREFERENCES → Themes in your Prestashop admin dashboard. Go to Logo tab to set up your logo.

Image Size Adjustment

In your Prestashop admin dashboard, navigate to: PREFERENCES > Images to configure the image size for category, manufacturer, supplier, product thumbnail, product popup, etc that fit your layout best.

1. Image size

 You can choose the type of product images that will suit your products the best. Images can be vertical rectangular (a form well suited for clothing), square (the best for shoes, accessories, electronics, etc.) or horizontal rectangular (furniture, auto, etc.).

Prestabrain Module Configuration

Pts Fashion Store supports a lot of Prestashop modules that help you build a basic Ecommerce functionality. Here is the instruction to use the extensions the way they are used in Pts Fashion Store demo site.

Pts Display Categories Information

The module allows you to show a list of categories in your website and the sum of products in each category. Banners of categories are edited in admin dashboard.

To configure the module, in your Prestashop admin dashboard, navigate to MODULES → Modules. Go to MODULES LIST section, search Pts Display Categories Information module then click on Configure.
